# Midwest Emo

**An interactive exploration of the music, architecture, and geography of the American Midwest.**

Built around 56 original interviews with musicians, producers, venue owners, architects, and scene participants — for Will Hagle's forthcoming book *Midwest Emo* (Bloomsbury, 2026).

## Concept

The website is the American Football house. You explore it room by room. Each room holds voices, memories, and fragments from the interviews. The architecture of the house is the architecture of the book.

**704 West High Street, Champaign, Illinois.**

## Structure

- `/` — The house from the street. The album cover. Just before snow.
- Basement → Living Room → Kitchen → Porch → Upstairs → Attic → Backyard
- Each room: interview quotes, ambient sound, thematic fragments
- Zoom out: house → neighborhood → Champaign → Midwest → the network

## Stack

Static HTML / CSS / JS. No framework. No build step.
